Introduction to quarto

Target Audience:

Master’s students with a basic understanding of R

Learning Objectives:

  • Understand what Quarto is and how it differs from R Markdown

  • Create and render basic Quarto documents in RStudio

  • Integrate R code, text, and visualizations into reproducible reports

  • Explore output formats and basic customization options

Part 1: Introduction and setup

  • What is Quarto?

    • Key features: multi-language support, reproducibility, flexible output formats

    • Use cases: presentation, report

  • Getting Started

    • Get Quarto https://quarto.org/docs/download/ (if needed)

    • Anatomy of a Quarto Document

      • Overview of RStudio integration and Quarto Visual Editor

    • Creating your first Quarto document (.qmd)

      • Open RStudio, create a new Quarto document (File > New File > Quarto Document)

      • YAML header: title, author, date, format

      • Markdown text: headings, lists, emphasis, links

      • Code chunks: inserting R code, chunk options and labels

      • Render button in RStudio, previewing output

      • Rendering to different formats: HTML, PDF, Word

  • Hands-on exercise 1:

    • create a quarto notebook

    • Edit YAML header, add some text, render to HTML
